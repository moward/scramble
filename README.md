Scramble
========

My CIS 120 final project. A platform word game. Fall 2013.

How to run
----------

1. Make sure you have JRE 7+ installed
2. Run the following from command line:

```Shell
cd ./bin/
java game/Game
```
  
How to play
-----------

Read the instructions [here](http://moward.com/apps/scramble/help/).
